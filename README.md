# IAFinanceCRM

Sistema de Controle Financeiro Pessoal e Empresarial.

## Stack
- **Frontend**: HTML5, TailwindCSS (CDN), JavaScript (Vanilla).
- **Backend**: PHP 7.4+ (PDO).
- **Banco de Dados**: MySQL.
- **Mobile**: Progressive Web App (PWA).

## Como Instalar
Leia o arquivo [walkthrough.md](walkthrough.md) (se disponível) ou siga os passos:
1. Importe `database.sql` no seu MySQL.
2. Configure `src/db.php`.
3. Rode em um servidor PHP.

## Recursos
- Dashboard com Gráficos (Chart.js)
- Gestão de Receitas e Despesas
- Suporte a Múltiplos Usuários
- Modo Escuro (Dark Mode) automático e manual
- Instalável no Celular
